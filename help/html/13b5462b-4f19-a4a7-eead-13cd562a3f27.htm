WhiteMathConstants E_SpigotAlgorithm Method whiteMath + whiteStructs - a programmer-friendly mathematical library
Runs a spigot algorithm that calculates the first digitsCount digits for the transcend constant e. The digits returned are of base decimalBase, which should be a power of ten: 10, 100, 1000 etc. REQUIREMENTS: 1. Memory consumption for the array of [n+safetyStorage] longs and the array of [n] ints.
