RSA Methods whiteMath + whiteStructs - a programmer-friendly mathematical library
The RSA type exposes the following members.
Methods
  NameDescription
Decrypt B, E 
Decrypts a long integer number using the RSA algorithm.
DecryptAsSingle B, E 
Treats a sequence of LongInt<B> numbers as a single big number which has been encrypted modulo publicKey and returns the result of decryption.
Encrypt B, E 
Encrypts a long integer number using the RSA algorithm. In case when the number to encrypt is bigger than the public key, its copy is continously divided by the key until it becomes zero, and remainders are encrypted.
GetKey B 
Creates a pair of two primes which, being multiplied one by another, produce a public key of desired length for the RSA algorithm.
GetSecretExponent B 
Calculates a public exponent on the basis of a pair of primes which form the RSA secret key and the number chosen as public exponent.
isPrimeOptimized B 
Скрытый метод. Перебирает сначала первые 100 простых чисел для проверки делимости. А уже дальше - Миллера-Рабина.
ToByteArray B 
Converts a LongInt number into a sequence of bytes.
ToLongInt B 
Converts a sequence of bytes into a LongInt number, e.g. to encrypt it using RSA algorithm.
