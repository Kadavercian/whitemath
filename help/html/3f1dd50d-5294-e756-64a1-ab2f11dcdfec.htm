SLAESolving LU_FactorizationSolving T, C  Method whiteMath + whiteStructs - a programmer-friendly mathematical library
This algorithm uses the LU-Factorization of coefficient matrix in order to calculate the solution of the equation system. As LU-Factorization is not guaranteed to exist for every non-singular matrix, this method may sometimes fail.
