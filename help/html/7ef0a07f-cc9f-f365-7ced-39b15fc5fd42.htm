sequence
Type: System.Collections.Generic IEnumerable T 
A calling sequence object.
centerValue
Type: T
A nonnegative value. Sequence elements that deviate from this value by more than allowedDeviationFactor will be filtered out.
allowedDeviationFactor
Type: System Double
The allowed factor of absolute deviation.
deviationType (Optional)
Type: whiteMath.Statistics DeviationType

            If deviation type is Downwards, values which are smaller than centerValue 
            by a factor of allowedDeviationFactor will be filtered.
            If deviation type is Upwards, values which are bigger than centerValue 
            by a factor of allowedDeviationFactor will be filtered.
            If deviation type is EitherSide, all values that differ from the centerValue
            by a factor of allowedDeviationFactor will be filtered.
