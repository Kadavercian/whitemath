getInBoundPoint
Method is used due to the existence of the milk area caused by 'indentFromImageBorder' parameter in the graphing argumens. If the point 'position' is out of the graphing area (in the "milk") this method would return the boundary point. Otherwise, the point returned is equivalent to the point passed. For example, the method is used by GraphicDrawer to determine if user has clicked an in-graph pixel, not the outer boundary of the image.
