Dif
Calculates the difference between two natural long integer numbers. WARNING: if the length of the bigger number is N, then the result digits array should be able to store N digits as well. No checking is performed, thus an IndexOutOfBoundsException can be thrown. We assume that number 'one' is more than number 'two'. If, nevertheless, two is bigger than one, the function would return true and the result would be equal to (BASE^result.Length - trueResultValue);
