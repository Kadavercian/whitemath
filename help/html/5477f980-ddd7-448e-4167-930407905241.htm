convertToListOfPairs T 
Makes an array of point objects from two arrays containing X coordinates and Y coordinates respectively. Make sure that no array is null and that they have the same length. Otherwise, a NullReferenceException or an ArgumentException will be thrown.
convertToMatrixRows T, C 
Converts a list of points to a matrix of size (Nx2) which rows are exactly the point coordinates.
convertToPairOfLists T 
Converts a list of points to a point which contains two single-dimensional arrays - an array of X's and an array of Y's.
