GraphingArgs Properties whiteMath + whiteStructs - a programmer-friendly mathematical library
The GraphingArgs type exposes the following members.
Properties
  NameDescription
BackgroundBrush
Gets or sets the brush which will be used to fill the background of the drawing area. If null, no background will be drawn.
CoordFont
Gets or sets the font which will be used to draw the values on the axes. If null, no values will be drawn on the axes.
CoordPen
Gets or sets the pen which will be used to draw the axes. If null, no axes will be drawn.
CurvePen
Gets or sets the pen which will be used to draw the graph curve. If null, the graph curve won't be drawn.
CurveType
Gets or sets the curve type of the graph.
GridPen
Gets or sets the pen which will be used to draw the coordinate grid. If null, no grid will be drawn.
IndentFromBounds
Gets or sets the indent from the boundaries of drawing area, in pixels.
