RandomLongIntRejection B  Class whiteMath + whiteStructs - a programmer-friendly mathematical library

This class provides high-quality uniform random LongInt<B> number generation, but is incredibly slow.

In practice, always use RandomLongIntModular<B> to achieve the same distribution quality with lower performance overhead.

Remarks

            even without modular reduction. Please use RandomLongIntModular<B>
