Parameters
minValue
Type: System Double
The absolute minimum of the graphing range. Must be less than the minimal value within current HistoGrapher's point-value pairs.
baseValue
Type: System Double
The absolute base-line of the graphing range. Columns' bottoms will have the coordinate of the baseline.
maxValue
Type: System Double
The absolute maximum of the graphing range. Must be bigger that the maximal value within current HistoGrapher's point-value pairs.
columnPortion
Type: System Double
A value from (0, 1] interval which specifies what percentage of width columns will take from automatically allocated.
G
Type: System.Drawing Graphics
A Graphics object to draw with.
drawingArea
Type: System.Drawing RectangleF
The area to contain the histogram. Full width of this rectangle will be used.
pointBrushes
Type: System.Collections.Generic Dictionary String, Brush 
A dictionary containing brushes for all HistoGrapher's points.
contourPen
