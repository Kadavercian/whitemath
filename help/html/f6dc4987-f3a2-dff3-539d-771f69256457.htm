Submatrix T, C  Class whiteMath + whiteStructs - a programmer-friendly mathematical library
The class represents the continuous submatrix of a numeric matrix. All matrix operations supported, but any changes made to the submatrix will be reflected on the parent matrix object. The mechanism uses an encapsulated parent matrix reference. An object of this class cannot be instantiated directly. Call method getSubmatrixAt() for a particular DoubleMatrix object instead.
