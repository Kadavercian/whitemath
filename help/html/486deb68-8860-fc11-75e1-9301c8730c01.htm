ServiceMethods Reorder_As_In_NodeList T  Method whiteMath + whiteStructs - a programmer-friendly mathematical library
Reorders the nodes within the list so that their order is the same as nodes order within the nodes list object passed. Requirement: the nodes list passed should contain unique nodes which are all the nodes of the linked list, not less and not more; This is automatically true if the nodes list passes the [!:IsObsolete()] test with checkOrder set to false.
