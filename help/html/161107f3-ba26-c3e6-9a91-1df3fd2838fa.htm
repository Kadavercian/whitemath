traverseInfix T (ITreeNode T , Int32)
Performs the infix traversing of the tree node and returns the list filled up with nodes in the traversion order. The prevIndex parameted is the index of child node after which the currently processed node should be traversed. If prevIndex is negative, the current node will be traverse first. Else, if it is more than or equals to the current node's children amount, the current node will be traversed last. Works only for any kind of trees.
