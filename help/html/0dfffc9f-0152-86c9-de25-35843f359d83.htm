HistoGraphingArgs Properties whiteMath + whiteStructs - a programmer-friendly mathematical library
The HistoGraphingArgs type exposes the following members.
Properties
  NameDescription
AutoValueLinePen
Gets pr sets the pen which will be used to draw lines going out from axis' automatic values and to the opposite border of the histogram.
AxisValueFont
A font which will be used to draw values on the axis.
ColumnBrushes
Gets or sets the dictionary that maps HistoGrapher's point names to brushes which will be used to draw corresponding columns on the histogram. If null, no columns will be drawn. This dictionary must contain all point names of HistoGrapher.
ColumnSeparatorPen
Gets or sets the pen which will be used to separate columns vertically from each other. If null, no separator will be drawn.
ColumnValueLinePen
Gets or sets the pen which will be used to connect the uppermost edge of the column with the axis (where the text specifying the column's value is usually located). If null, no such lines will be drawn.
IncrementDirection
Gets or sets the orientation of the histogram, that is, the increment direction of the columns.
InsideColumnValueFont
A font which will be used to draw values inside the columns. If null, no values will be drawn inside the column.
InsideColumnValuePosition
A value indicating the alignment of values inside the columns. (if these values are drawn at all).
PointAliases
Gets or sets the dictionary that maps HistoGrapher's point names to aliases which will instead be drawn on the histogram. If null, existing point names will be used.
