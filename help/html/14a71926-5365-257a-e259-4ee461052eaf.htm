ICalc T  Methods whiteMath + whiteStructs - a programmer-friendly mathematical library
The ICalc T  type exposes the following members.
Methods
  NameDescription
decrement
Decrements a number so that a := a - fromInt(1). This method requires implementation because in many cases the decrement operation can be implemented much more effective than by just substracting fromInt(1) from the current value.
dif
Returns the difference of two numbers.
div
Returns the first number divided by the second number.
eqv
Tests if the first number is equal to the second. Should be symmetric: a==b must be equivalent to b==a.
fromDouble
Creates a number of T type from a double value.
fromInt
Creates a number of T type from an integer value.
getCopy
Returns an independent copy of the number.
increment
Increments a number so that a := a + fromInt(1). This method requires implementation because in many cases the increment operation can be implemented much more effective than by just adding fromInt(1) to the current value.
intPart
Returns the integer part for the number. Should have the same sign as the number itself.
isEven
Returns true if the integral number is even, that is, has no remainder of division by two.
isNaN
Tests whether the value is not a number (NaN) value. If the type does not support NaN's, this method should always return false.
isNegInf
Tests whether the value is a negative infinity. If the type does not support infinity values, this method should always return false.
isPosInf
Tests whether the value is a positive infinity. If the type does not support infinity values, this method should always return false.
mor
Tests if the first number is more than the second.

(mor(one, two)) means that - but not equivalent to - : (!mor(two, one))

mul
Returns the multiplication product of two numbers.
negate
Negates the number so that:
  1. a := -a
  2. a + (-a) = 0.
parse
Creates a number of T type from a string representing the resulting value.
rem
Returns the integral remainder of number division.
sum
Returns the sum of two T numbers.
