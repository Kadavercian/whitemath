The interval on which the integral will be approximated.
rectangleHeight
Type: T
The height of the testing rectangle. Should be more than the function's absolute maximum on the interval tested, otherwise the method would return wrong results. On the other side, the difference between the height and the function's absolute maximum should not be very large as it will reduce the accuracy of the method. The ideal case is equality of the max f(x) on [a; b] and the rectangle height.
throwsCount
Type: T
A positive integer value of overall tests.
