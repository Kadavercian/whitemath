BaseDivide
If the digits base for the current number X is BASE, the method would return return X / BASE ^ basePower. Equivalent to removing basePower from the beginning of the number, i.e. shifting right by basePower BASE-based digits. Is quick for any digit base.
BaseMultiply
If the digits base for the current number X is BASE, the method would return return X * BASE ^ basePower. Equivalent to adding basePower empty digits to the beginning of the number, i.e. shifting left by basePower BASE-based digits. Is quick for any digit base.
BaseRootDivide
If the base of the current number digits is an integer power of some value rootValue, then this method, provided with information on which power it is exactly (rootDegree), will return the current number integrally divided by rootValue^multiplyByRootPower. Is QUICK for any digit bases. Takes as much as O(n) time to perform. For example, if the base of number digits is 125, you can easily perform a division by 25, because the base is a power of five, and 25 is 5^2. Thus the result would be [current num] / 5^2 - and calculated quickly. WARNING: the method will return totally wrong values if the base of current number's digits is not an exact integer power of rootValue.
BaseRootMultiply
If the base of the current number digits is an integer power of some value rootValue, then this method, provided with information on which power it is exactly (rootDegree), will return the current number multiplied by rootValue^multiplyByRootPower. Is QUICK for any digit bases. Takes as much as O(n) time to perform. For example, if the base of number digits is 125, you can easily perform a multiplication by 25, because the base is a power of five, and 25 is 5^2. Thus the result would be [current num] * 5^2 - and calculated quickly. WARNING: the method will return totally wrong values if the base of the digits for current number is not an exact integer power of rootValue.

DecimalDivide
Shifts the current number to the right by the specified amount of decimal digits. Equivalent to division by 10^tenPower. IS quick only if the base of current number's digits is a power of ten. Otherwise, simple powering and multiplication operations shall be performed. Implicitly calls the method BaseRootDivide(Int32, Int32, Int32) with parameters rootValue = 10, rootDegree = log10(BASE), divideByRootPower = tenPower.
DecimalMultiply
Shifts the current number to the left by the specified amount of decimal digits. Equivalent to multiplication by 10^tenPower. WARNING! IS quick only if the base of this number is a power of ten. Otherwise, a LongInt(B) of current base will be first created as a result of integer powering 10^tenPower, and further multiplication shall be performed. Implicitly calls the method BaseRootMultiply(Int32, Int32, Int32) with parameters rootValue = 10, rootDegree = log10(BASE), multiplyByRootPower = tenPower.
