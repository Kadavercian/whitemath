OtherExtensions GetReverseComparer T  Method whiteMath + whiteStructs - a programmer-friendly mathematical library
Returns the reverse comparer for an IComparer() object. For example, each time the incoming comparer would tell that object A is 'more' than object B (returns positive value), this comparer will tell the opposite and return the same absolute value, but with negative sign.
