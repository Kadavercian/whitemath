LongInt B  CreateRandom_Binary Method whiteMath + whiteStructs - a programmer-friendly mathematical library
The constructor designed to create a pseudo-random LongInt number, using an integer amount of BINARY digits (bits) specified by the user. For example, if the BASE of the current number is 256, and binaryDigitCount is 9, the number would contain two 256-based digits.
