SturmPolynomSequence
Returns the Sturm polynom sequence for the current polynom. Used in Sturm theorem which helps to find the number of real roots contained in some interval. nrr(f(x), x in (a; b)) = V(S(a)) - V(S(b)); Where V(S(x)) is number of sign changes in the polynom's Sturm sequence. Zeroes of the sequence do not count. For example, if the Sturm sequence at some point 'x' produces values -1, 1, 0, 1, then V(S(x)) = 1.
