ServiceMethods GetNodes T  Method whiteMath + whiteStructs - a programmer-friendly mathematical library
Returns the array of linked list nodes, starting with the first and ending with the last. Takes O(N) time. Warning! The array may go to obsolete/inconsistent state, stopping to represent the full collection of linked list nodes in their supposed order, if the list object is changed after or during getting the nodes list. (e.g. nodes are reordered, added, removed etc.) Consider using IsObsolete() with checkOrder parameter equal to true for such an array, if you are in doubt.
